# Amazon Sales Analytics Dashboard (Power BI)

## Project Overview
This project analyzes Amazon sales data to understand revenue trends, order distribution, customer demand, and regional performance.

## Tools Used
- Power BI
- Excel (Data Source)

## Key KPIs
- Total Sales Amount
- Total Quantity Sold
- Total Orders
- Shipping Coverage (Postal Codes)

## Dashboard Features
- Interactive slicers for Category and Order Status
- Order distribution by delivery status
- State-wise and city-wise sales analysis
- Category-wise quantity and revenue insights
- B2B vs Retail order comparison

## Key Insights
- Retail (B2C) orders dominate overall sales
- Maharashtra and Karnataka generate the highest order volume
- Sets and Kurtas are the top-performing categories
- Order cancellations are relatively low

## Outcome
This dashboard helps stakeholders monitor sales performance, identify high-performing regions and products, and support data-driven decision-making.
